# Loan Risk Prediction

## Overview

This project develops machine learning models to predict loan default risk using financial and credit-related data. The goal is to support risk assessment and improve lending decisions through predictive analytics.

## Dataset

- Financial and credit-related customer data
- Target: Loan default prediction

## Technologies

- Python
- Scikit-learn
- Pandas
- NumPy

## Methods

- Data Preprocessing
- Feature Engineering
- Feature Selection
- Classification Models
  - Decision Tree
  - Random Forest
  - K-Nearest Neighbors (KNN)

## Evaluation Metrics

- Precision
- Recall
- F1-Score
- Accuracy

## Key Features

- Data cleaning and preprocessing
- Feature engineering
- Model comparison and evaluation
- Loan default risk prediction
